
LIVRO: Concise Guide to Object-Oriented Programming, Kingsley Sage
OBJETIVO DO PROJETO: Estudar um livro de java, revisar fundamentos
VERSÃO ou DATA: Início da leitura: 26/08/2022, conclusão em 31/08/2022
COMO INICIAR O PROJETO: Foi utilizado o BlueJ, recomendação do livro
OBS.: BlueJ não é um editor profissonal, mas possui uns recursos que ajudam no entendimento do projeto.
Só utilizei por causa do livro, em um projeto profissional é provável que não o utilizarei.
PARTE BOA: faz uma exibição visual do projeto com o relacionamento entre as classes.
Permite executar um método de uma classe sem ter que fazer a instância do objeto ser chamada em um main().
Permite inspecionar o objeto, mostrando os seus valores. O escopo fica destacado, muito bom pra codificação extensa.
Utilizaria ele como introdução para dar aula de programação em Java.
PARTE RUIM: o autocompletar é limitado, não dá pra navegar entre objetos, vc precisa achar o arquivo e abrir. 
Não tem facilitadores de criação de código. A criação de pacotes é um pouco confusa de início. 
Debugar também é complicado, não consegui visualizar os valores de objetos/variáveis.
